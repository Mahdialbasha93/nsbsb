Help Guide

Getting Your Token

1. Open Discord in your browser

2. Press F12 to open Developer Tools

3. Go to Application → Local Storage

4. Find and copy the token value

Getting Server IDs

1. Open Discord Settings

2. Go to Advanced → Enable Developer Mode

3. Right-click on any server → Copy ID

Auto-Reconnect Feature

• Automatically reconnects if cloning slows down

• Resumes from where it left off

• Adjustable threshold and retry attempts

Keyboard Shortcuts

Ctrl + Enter - Start/Stop cloning

Ctrl + L - Clear logs

Ctrl + S - Save settings
